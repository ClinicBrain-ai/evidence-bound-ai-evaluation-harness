# Safety Boundary

This project is not for clinical use.

It does not:

- Diagnose.
- Recommend treatment.
- Validate clinical correctness.
- Assess medical safety.
- Provide patient-facing medical advice.
- Replace clinicians or licensed reviewers.
- Claim HIPAA compliance.
- Claim production readiness.
- Function as a medical device.
- Use real patient data.

The project evaluates the relationship between AI-generated healthcare-adjacent text and bounded source evidence. It is a portfolio-grade prototype for reviewability and evidence-alignment evaluation only.
