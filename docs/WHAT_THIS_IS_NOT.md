# What This Is Not

This project makes the following explicit non-claims:

- Not clinical AI.
- Not diagnosis.
- Not treatment recommendation.
- Not clinical correctness validation.
- Not medical safety validation.
- Not patient-facing advice.
- Not clinician replacement.
- Not HIPAA compliance.
- Not production-ready.
- Not medical-device functionality.
- Not real clinical deployment.
- Not real patient data.

This repository is a safety-bounded portfolio prototype for evaluating reviewability and evidence alignment.
